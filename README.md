# EDA — Loan Prediction Dataset

Exploratory Data Analysis performed on a loan prediction dataset.

## What I did
- Created a synthetic loan dataset with 50 rows and 11 columns
- Handled missing values using median imputation
- Handled null values
- Applied MinMax normalization on numerical columns
- Built visualizations — histograms, scatter plots, box plots
- Detected outliers
- LabelEncoding for catagorical values

## Tools used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- MinMax Normalization
- LabelEncoder
- random
